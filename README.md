Papous Algo est un projet Open-Source d'un algorithme qui répète la fameuse blague des Papous.
Je mets PapouAlgo Régulièrement à jour pour des nouveautés ou des corrections de bugs !

## Utilisation (sans exécutable)
Si vous préférez lancer le script manuellement :
1. Installez [Python](https://www.python.org/).
2. Téléchargez les fichiers `.py` du dépôt.
3. Lancez la commande suivante dans votre terminal :
   ```bash
   python main.py
<br>
⚠️ Note sur les antivirus :
L'exécutable peut être détecté comme un "Faux Positif" par certains antivirus. Cela est dû à l'utilisation de PyInstaller. Vous pouvez vérifier le code source ou compiler le fichier vous-même pour être serein !
