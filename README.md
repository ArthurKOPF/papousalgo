PapouAlgo est un projet Open-Source d'un algorithme qui répète la fameuse blague des Papous.
Je met PapouAlgo Régulièrement à jour pour des nouveautées ou des corrections de bugs !

⚠️ Note sur les antivirus :
L'exécutable peut être détecté comme un "Faux Positif" par certains antivirus. Cela est dû à l'utilisation de PyInstaller. Vous pouvez vérifier le code source ou compiler le fichier vous-même pour être serein !
